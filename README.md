# archive-portal-global

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容相互独立，不针对特定域名或网站，旨在长期保存和提供静态页面访问。

## 项目简介

archive-portal-global 是一个静态页面归档仓库，收集并托管各类独立的 HTML 文档。所有页面均以原始 HTML 格式存储，便于直接通过浏览器访问或嵌入其他应用。

## 目录说明

```
.
├── pages/          # 存放所有归档的 HTML 页面文件
├── assets/         # 页面依赖的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的归档页面，命名规则见内部说明。
- `assets/`：按需存放页面所需的公共或私有资源。

## 页面归档说明

- 每个页面均为完整、自包含的 HTML 文档（含 `<head>` 和 `<body>`）。
- 页面内容不包含任何推广、诱导点击或商业营销信息。
- 归档页面可能包含外部引用的资源（如字体、图标库），但尽量使用本地备份或 CDN 稳定链接。
- 页面文件名通常反映内容主题，但无具体域名或网站关联。

## 维护说明

- 本仓库由维护者不定期更新，添加新的归档页面或修复现有页面兼容性问题。
- 欢迎通过 Issues 提交页面归档建议或报告页面显示异常。
- 不接受包含恶意代码、侵权内容或违反 GitHub 服务条款的页面提交。
- 页面内容仅供存档和参考，不保证其适用于所有浏览环境。

## 使用方式

直接通过 GitHub Pages 或克隆仓库到本地，用浏览器打开 `pages/` 下的 HTML 文件即可访问。

## 许可

本仓库内容遵循相应页面的原始许可协议。若无明确说明，默认采用 MIT 许可证。
